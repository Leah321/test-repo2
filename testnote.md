## How to push files to gitHub
### My note

* $git clone "repo address url"
* $git remote -v
* $touch new.md
* $git status
* edit new.md using notepad and save
* $git add .
* $git status
* $git commit -m "edit readme and add new.md"
* $git status
* $git log
* $git push origin master
* log in your account and check the changes
